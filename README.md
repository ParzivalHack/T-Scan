### ABOUT :

Ping, scan and search for vulnerabilities with this nmap-based shell script.

### TESTED ON :

* Termux✅
* Linux✅

### REQUIREMENTS :

* nmap [pkg install nmap -y]

### INSTALLATION & USAGE:

* git clone https://github.com/ParzivalHack/T-Scan
* cd T-Scan
* bash T-Scan.sh

### FEATURES :

* [+] Ping website!
* [+] Scan website!
* [+] Search for vulnerabilities!
* [+] Beginner friendly!
* [+] Works on Termux!
